# SETUP #
```
$ curl https://install.meteor.com/ | sh
$ git clone https://github.com/Falieson/planetx-boilerplate.git
$ cd planetx-boilerplate
$ ln -s .meteor/packages meteor_packages && ls -lsa
$ meteor npm i
$ meteor
```
